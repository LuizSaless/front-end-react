Introdução ao HTML

O que é HTML?

HTML significa HyperText Markup Language. Ele é a linguagem usada para estruturar páginas na web, permitindo organizar textos, imagens, links e outros elementos.

Principais Tags de Formatação

<b> — Deixa o texto em negrito.

<strong> — Destaca o texto com importância semântica: este texto é importante.

<i> — Deixa o texto em itálico: este texto está em itálico.

<em> — Dá ênfase semântica ao texto: este texto tem ênfase.

<mark> — Marca o texto: este texto está marcado.

<small> — Deixa o texto menor: texto pequeno.

<del> — Texto deletado: este texto foi removido.

<sub> — Texto subscrito: H2O

<sup> — Texto sobrescrito: x2
