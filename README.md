## Fundamentos do Desenvolvimento Front-End

Este repositório reúne a **resolução prática de exercícios introdutórios de HTML e CSS**, que fazem parte do módulo **Fundamentos do Desenvolvimento Front-End**, base essencial do **Front-end com ReactJS**.

O foco é consolidar os conceitos fundamentais do desenvolvimento web antes do uso de bibliotecas e frameworks modernos.

---

## Conteúdos Abordados

### HTML – Conceitos Básicos

* Estrutura básica de um documento HTML
* Tags essenciais (`html`, `head`, `body`)
* Organização e boas práticas de código

### Principais Tags de Formatação

* Títulos (`h1` a `h6`)
* Parágrafos (`p`)
* Ênfase e destaque (`strong`, `em`, `b`, `i`)
* Quebras de linha e separadores (`br`, `hr`)

### Imagens, Links e Listas

* Inserção de imagens (`img`)
* Links internos e externos (`a`)
* Listas ordenadas e não ordenadas (`ol`, `ul`, `li`)

### Tabelas, Atributos e HTML Semântico

* Estruturação de tabelas (`table`, `tr`, `td`, `th`)
* Uso de atributos HTML
* Introdução ao HTML semântico (`header`, `nav`, `main`, `section`, `article`, `footer`)

---

## CSS – Estilização Básica

### Tipos de CSS

* CSS Inline
* CSS Interno
* CSS Externo

### Conceitos Fundamentais

* Unidades de medida (`px`, `%`, `em`, `rem`)
* Background (cores e imagens)
* Box Model (content, padding, border e margin)
* Estilização de Links, listas e tabelas
* Position, Pseudo Classes e Pseudo Elementos 

---

## Objetivo do Projeto

Projeto de caráter **educacional**, indicado para:

* Iniciantes em desenvolvimento Front-End
* Estudantes de React que precisam reforçar a base
* Revisão de conceitos fundamentais de HTML e CSS

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* Visual Studio Code

Os exercícios foram resolvidos de forma progressiva, preparando o ambiente de aprendizado para conteúdos futuros como **JavaScript, React, layout responsivo, Flexbox e Grid**.
